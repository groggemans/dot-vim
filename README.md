# dot-vim
My vim configuration

Feel free to use it as a starting point for your own vim configuration!

## Install
**Requirements:** [Ellipsis][ellipsis] `(>=v1.9.0)`

``` shell
$ ellipsis install groggemans/dot-vim
```
